module Tag_parser: sig .. end
Functions for parsing individual Liquid template tags
val extract_tag : start:string ->
       stop:string -> Astring.String.sub -> Astring.String.sub option
extract_tag ~start ~stop s will extract a substring that represents the text in "<start><ws><text><ws><stop>". Whitespace is trimmed, and None is returned if non-empty text could not be parsed.
type lines = Astring.String.Sub.t list
lines represents a list of lines that are substrings from a larger string.
